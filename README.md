# AI Text Summarizer

AI-powered text summarization system built using FastAPI and Hugging Face Transformers.

## Features
- Transformer-based text summarization
- FastAPI backend integration
- HTML/CSS frontend
- T5 Transformer model
- Real-time summary generation

## Technologies Used
- Python
- FastAPI
- Hugging Face Transformers
- PyTorch
- HTML/CSS/JavaScript
